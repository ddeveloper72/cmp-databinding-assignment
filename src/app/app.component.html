Bind it!

  1. Create three new components: GameControl, Odd and Even
  2. The GameControl Component should have buttons to start and stop the game
  3. When starting the game, and event (holding a increment number) should get emitted each second (re = setInterval())
  4. The event should be listenable from outside the component
  5. When stopping the game, no more events should be emitted (ClearInterval(ref))
  6. A new Odd number component should be created for every odd number emitted, the same should happen for the Even Component (on even numbers)
  7. Simply output Odd - Number or Even - Number in the two components
  8. Style the element (eg paragraph) holding your output text differently both components.
